Coffee Lover Project

Welcome to your bootstrap project: Coffee Lover

You will recreate this webpage and an additional Form Page

Just folow the instructio step here to re-create these webpages!

  1. Create a general landing page with a jumbotron
  2. Next craete a nav bar on the page that links to another HTML files.
  3. Next use the grid system to add in thumbnail pictures of cofee.Here are the images:
    • One

  4. Next make sure that at the largest seetings you have 3 coloumns f three pictures. on the smallest screen settings you should have 2 column.
  5. Next create another html file for the sign in page.
  6. Create a form to sign up with email, password, and a check box
